Politique de Confidentialité

Dernière mise à jour : Décembre 2024

Résumé : ikaConfort respecte votre vie privée. Nous collectons uniquement les données nécessaires au fonctionnement de l'application et ne vendons jamais vos informations personnelles.

1. Introduction

Bienvenue sur ikaConfort. Cette politique de confidentialité explique comment nous collectons, utilisons, partageons et protégeons vos informations personnelles lorsque vous utilisez notre application mobile.

En utilisant ikaConfort, vous acceptez les pratiques décrites dans cette politique.

2. Données Collectées

2.1 Données que vous nous fournissez

  • Informations de compte : nom, adresse e-mail, numéro de téléphone
  • Informations de profil : photo de profil (optionnel)
  • Communications : messages avec les agents immobiliers

2.2 Données collectées automatiquement

  • Données de localisation : avec votre permission, pour afficher les propriétés à proximité
  • Données d'utilisation : propriétés consultées, recherches effectuées, favoris
  • Données techniques : type d'appareil, version de l'OS, identifiants uniques

2.3 Données provenant de tiers

  • Données d'authentification via Firebase Authentication

3. Utilisation des Données

Nous utilisons vos données pour :

  • Fournir et améliorer nos services immobiliers
  • Personnaliser votre expérience (recommandations de propriétés)
  • Faciliter la communication avec les agents
  • Gérer vos réservations et rendez-vous
  • Envoyer des notifications pertinentes (avec votre consentement)
  • Assurer la sécurité de l'application
  • Respecter nos obligations légales

4. Partage des Données

Nous ne vendons jamais vos données personnelles. Nous pouvons partager vos données avec :

  • Agents immobiliers : uniquement les informations nécessaires pour traiter vos demandes
  • Prestataires de services : Firebase (Google) pour l'hébergement et l'authentification
  • Autorités légales : si requis par la loi

5. Stockage et Sécurité

Vos données sont stockées de manière sécurisée sur les serveurs Firebase (Google Cloud Platform) avec :

  • Chiffrement des données en transit (TLS/SSL)
  • Chiffrement des données au repos
  • Contrôles d'accès stricts
  • Audits de sécurité réguliers

6. Vos Droits (RGPD)

Conformément au RGPD, vous avez le droit de :

  • Accès : obtenir une copie de vos données personnelles
  • Rectification : corriger vos données inexactes
  • Effacement : demander la suppression de vos données
  • Portabilité : recevoir vos données dans un format structuré
  • Opposition : vous opposer au traitement de vos données
  • Limitation : limiter le traitement de vos données

Pour exercer ces droits, utilisez la fonction "Exporter mes données" ou "Supprimer mon compte" dans l'application, ou contactez-nous.

7. Conservation des Données

  • Données de compte : conservées tant que votre compte est actif
  • Données de réservation : 3 ans après la dernière activité
  • Données de navigation : 12 mois

Après suppression du compte, vos données sont effacées sous 30 jours.

8. Cookies et Technologies Similaires

L'application utilise des technologies de stockage local (AsyncStorage) pour :

  • Mémoriser vos préférences
  • Améliorer les performances
  • Maintenir votre session connectée

9. Mineurs

ikaConfort n'est pas destiné aux personnes de moins de 18 ans. Nous ne collectons pas sciemment de données concernant des mineurs.

10. Modifications

Nous pouvons mettre à jour cette politique. En cas de modifications significatives, nous vous en informerons via l'application ou par e-mail.
